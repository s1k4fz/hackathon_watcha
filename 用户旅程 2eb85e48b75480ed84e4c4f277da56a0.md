# 用户旅程

### **整体结构（总时长约5分钟）**

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [开场]  →  [相遇]  →  [首战]  →  [对话]  →  [决战]  →  [尾声]   │
│    30秒      1分钟      1.5分钟     30秒      1.5分钟     30秒    │
│                                                             │
│   世界观     认识绫雀    教学战斗    建立羁绊    展示深度    留下悬念  │
│   铺垫       展示性格    核心玩法    角色魅力    技能连携    想玩更多  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### **每个节点的设计**

### **1. 开场（30秒）**

```
【黑屏 + 文字】（3秒）
"2087年，静默灾厄后的第三年。"
"你是一名拾荒者，在废墟深处发现了一个沉睡的身影——"

【画面】渐入绫雀的形象，（画面视角从闭眼到睁眼）

【黑屏】
"……"
"……能听到吗？"
"喂，醒醒。"

【画面渐入】
模糊的视野中，一个银发少女的脸逐渐清晰——

绫雀："终于醒了？还以为你要这样睡过去了。"
（后退一步，打量玩家）
绫雀："你的意识波动很微弱，但……居然还活着。运气不错。"
```

**目的**：建立氛围，最小成本交代世界观

---

### **2. 相遇/唤醒（1分钟）**

```
绫雀："你知道自己是谁吗？知道这是哪吗？"

【玩家输入框】
提示：试着回应她（可以说"不知道"、"我记不起来了"、或任何内容）

绫雀：（根据玩家输入反应）

【如果玩家表示不知道/失忆】
绫雀："……记忆损伤吗。静默灾厄的后遗症，不稀奇。"
"简单说吧——现在是2087年，三年前一场灾难摧毁了大部分的意念链接系统。"
"你现在在晓光机关的废墟里。至于你为什么会在这……我也不知道。"

【如果玩家瞎说/试探】
绫雀："……你在说什么？脑子也坏掉了？"
（叹气）"算了，反正我也没指望从你这得到什么有用的信息。"

绫雀："我是残响体绫雀，编号 LQ-07。"
"奇怪的是……我能感应到你的意念波动。"
"你可能是个『协调者』——如果是真的，那你可就值钱了。"
```

**目的**：

- 自然引入"自然语言交流"机制
- 第一次展示 LLM 的反应能力
- 建立玩家与绫雀的初始关系

---

### **3. 首战/教学战（1.5分钟）**

```
【警报声】
绫雀："——有东西来了。"
（武器展开）
绫雀："躲到我身后。不对，等等……"

（看向玩家）
绫雀："你身上有协调者的波动。也许你能指挥我。"
"我的链接系统受损，没法接收标准协议。"
"但如果你用**你自己的话**告诉我战术意图——也许能绕过去。"

"试试看。告诉我该怎么打。"

【进入战斗】
教学敌人出现
引导玩家尝试自然语言指令

【战斗中的反馈】
绫雀："哦？这种模糊的指令我居然能理解……有意思。"
绫雀："你的战术素养很烂，但直觉不错。"

【胜利】
绫雀："……成功了。"
（若有所思）
绫雀："你真的是协调者。而且是能和我产生共鸣的那种。"
"这太巧了……或者说，太不巧了。"
```

**目的**：核心玩法首次完整展示

---

### **4. 幕间对话（30秒）**

```
绫雀："老实说，我本来只是在这里避难。"
"没想到会捡到一个失忆的协调者。"

【玩家输入框】
提示：你可以问她任何问题

绫雀：（根据问题回答）

【如果问"你是什么"】
绫雀："残响体。用死人的意识碎片造出来的战术人格。"
"……别用那种眼神看我。我已经习惯了。"

【如果问"你为什么救我"】
绫雀："……谁说我救你了。只是顺手而已。"
（移开视线）
"而且……协调者很稀有。你对我有利用价值。别会错意。"

绫雀："外面很危险，你一个人活不下去。"
"正好我也需要一个能指挥我的人。"
"我们……暂时合作吧。"
```

**目的**：展示角色深度、建立情感连接

---

### **5. 决战/Boss战（1.5分钟）**

```
【移动一段距离后，强敌出现】
绫雀："停——这个信号……"
（表情变得严肃）
绫雀："『暗影骑士』级别的战斗单位。比刚才那些杂鱼强多了。"

（看向玩家）
绫雀："你确定要打？如果你跑的话，我可以断后——"

【玩家输入框】
提示：告诉她你的决定

绫雀：（根据玩家选择）

【如果玩家选择一起战斗】
绫雀："……哼。"
（嘴角微微翘起）
绫雀："行吧。第一天就想当英雄？"
"不讨厌这种疯劲。来吧，搭档。"

【进入Boss战】
更强的敌人
需要更有策略的指令
战斗中展示大招

【胜利】
绫雀："呼——"
（难得地笑了一下）
绫雀："不错嘛，你这个失忆的家伙。"
"也许……跟着你不会太无聊。"
```

**目的**：展示战斗深度、玩家有成就感

---

### **6. 尾声（30秒）**

```
绫雀："这附近应该暂时安全了。"
"我们需要离开这片废墟，找到补给和情报。"

（停顿）
绫雀："我感应到了……其他残响体的信号。"
"也许她们知道关于这场灾厄的更多信息。"
"也许……能帮你找回记忆。"

（看向玩家）
绫雀："你问我为什么要帮你？"
"……"
"也许只是因为，你是第一个把我当'绫雀'而不是'LQ-07'的人。"

【画面渐暗】
"—— Chapter 1: Awakening ——"
"—— 第一章：醒来 ——"

【显示】
"感谢体验「残响：叙语连接」"
"你与绫雀的故事，才刚刚开始……"

【画面渐暗】
"—— 第一章 完 ——"

【显示】
"感谢体验「残响：叙语连接」Demo"
"完整版敬请期待"

【选项】
[再玩一次]  [返回主界面]  [查看角色档案]
```

**目的**：给故事画上句号、留下悬念、暗示更多内容
